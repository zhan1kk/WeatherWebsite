# WeatherWebsite

## Simple weather website

This website was developed as a part of the Frontend Programming final exam. 

## Features
- Responsive Website
  - Display of current weather
  - Forecast for the next week
  - Location search functionality

## Technologies Used

- HTML5
- CSS3
- Bootstrap
- JavaScript/ES6
- OpenWeatherMap API
- Chart.js
